#  jstat命令查看jvm的GC情况 （以Linux为例）

 

 jstat命令可以查看堆内存各部分的使用量，以及加载类的数量。命令的格式如下：

 jstat [-命令选项] [vmid] [间隔时间/毫秒] [查询次数]

 **注意！！！：使用的jdk版本是jdk8.**

 

##   类加载统计：

　![img](assets/1139681-20171030173947871-420843554.png)

- **Loaded:**加载class的数量
- **Bytes：**所占用空间大小
- **Unloaded：**未加载数量
- **Bytes:**未加载占用空间
- **Time：**时间

 

##   编译统计

　![img](assets/1139681-20171030174025668-1000221849.png)

- **Compiled：**编译数量。
- **Failed：**失败数量
- **Invalid：**不可用数量
- **Time：**时间
- **FailedType：**失败类型
- **FailedMethod：**失败的方法
